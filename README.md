# Civilizations — 3D Graph Demo

An interactive 3D knowledge graph of ancient civilizations — polities, rulers,
places, and their succession — rendered as a single self-contained HTML file.

**View it live:** https://creatorinla.github.io/civilizations-graph-demo/

Generated by AutoWiki, a deterministic, citation-backed knowledge-graph and
wiki engine. The graph data comes from Wikidata and curated historical texts.
Node colors encode entity type, line widths encode claim strength, and
animated particles trace flow along succession edges. Click a node for
details; use the search box and legend to filter.

Everything — data, renderer, and libraries — is embedded in `index.html`.
No external requests; it works offline.

Built with [3d-force-graph](https://github.com/vasturiano/3d-force-graph)
and [three.js](https://threejs.org) (MIT).
